Conditional Logic - Loops

Exercise 1: Console log "Hello" 7 times

Exercise 2: Console log the numbers 1-15 inclusive

Exercise 3: Console log 1 then every third number up to 15 inclusive.

Exercise 4: Console log every third number between 1-15 inclusive

Exercise 5: Console log the numbers 15-1 inclusive

Exercise 6: Console log every third number between 15-1 inclusive except 9

Exercise 7: Console log "Hello" 5 times, then "Goodbye" twice then "Hello" 3 more times

Exercise 8: Ask the user for a number between 1 and 100, then console log every number up the user's number, inclusive

Exercise 9: Ask the user to enter a Nordic country, and display that country's capital city. Use a switch

Exercise 10: The Fibonacci Sequence is a mathematical formula dating back at last 2,500 years

Each number in the sequence is the sum of the two preceding numbers. Eg 0 1 1 2 3 5 8 13 etc

Starting with 0 and 1, console.log the Fibonacci numbers up to & including 144

Exercise 11: Console the letters A, B and C and numbers 1-10 in this pattern: A1, A2, A3 etc up to A10, B1, B2 etc up to B10, C1 etc

Exercise 12: Write some code that produces: a0 a2 a3 x0 a0 a2 a3 x1 x2 a0 a2 a3 x3

Exercise 13: Write a guessing game that asks a user for a number between 1-100 inclusive, and alerts "Higher" or "Lower" until the user guesses the correct number.

When the user guesses the correct number alert "You Win!"

Exercise 14: Redo Exercise 10, but also show the previous guesses the user has made on each failed guess
